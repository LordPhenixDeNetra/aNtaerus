# aNtaerus Workspace

Le README public du projet est maintenant à la racine du dépôt :

- [README.md](../README.md)

Ce sous-répertoire `antaerus/` contient le workspace applicatif principal du monorepo.

## Contenu

- `docs/` : documentation technique interne au workspace
- `engine/` : orchestration transverse
- `interfaces/` : interface Web et gateway Go
- `kernel/` : contrats, schémas et primitives partagées
- `providers/` : services Python et Rust
- `scripts/` : scripts de démarrage, d'arrêt et de validation
